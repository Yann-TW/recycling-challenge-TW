# Tradeweb Recycling Challenge V4

A self-contained static website. Players only need the published URL; they never download these files.

## Folder structure

- `index.html` — webpage entry point (must remain at the root)
- `assets/css/styles.css` — visual design
- `assets/js/items.js` — 100-item question pool
- `assets/js/app.js` — game logic
- `assets/img/network-grid.svg` — local background artwork

## Quick local check

Double-click `index.html`. The game has no external dependencies.

## Publishing

See `PUBLISHING-GUIDE.md`. Upload the entire `recycling-game-v4` folder to a static website host. Share the final `https://...` site URL, not this folder or ZIP.
